# wandasystems-site

> WandaSystems Company Website — Built with Astro 4 and Tailwind CSS.

This is the central public face of **WandaSystems**, designed for high-performance delivery, clean AI-first architecture, and accessibility.

## 🚀 Tech Stack

- **Framework**: [Astro 4](https://astro.build/) (Static Site Generation / Hybrid)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **UI Components**: React (Islands Architecture)
- **Deployment**: GitHub Pages / Custom VPS

## 🛠️ Getting Started

### Prerequisites

- Node.js (v18.0.0 or higher)
- npm / pnpm

### Installation

```bash
# Clone the repository
git clone https://github.com/wanda-OS-dev/wandasystems-site.git

# Install dependencies
npm install

# Start development server
npm run dev
```

### Building for Production

```bash
npm run build
```

The output will be generated in the `dist/` directory.

## 🛡️ Security & Performance

- **Sentinel**: Automated security audits and input validation.
- **Bolt**: Performance optimizations including image compression and lazy hydration.
- **Palette**: UX/Accessibility standards (ARIA compliant).
